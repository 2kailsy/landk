## ✨ 开发
#### 📦 部署
1.前置环境
- PHP 7.0.0+
- MySQL 5.0+

2.配置
- 克隆项目到本地
- 创建数据库，导入SQL脚本
- 去到`\config`文件夹下找到`database.php`配置`数据库账户密码`
- 访问网站
